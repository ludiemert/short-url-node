### Short Links
Short links is a simple project to practice server creation with Node and Fastify

<b>Functionalities:</b>
- Get links
- Get metrics
- Go to URL and save access
- Create Links

## 🔧 How to execute it 
Go to the folder where you clone the project, open a terminal and follow these steps:
  Client:
  - 01 - cd client
  - 02 - npm i
  - 03 - npm run dev

  Server:
  - 01 - cd server
  - 02 - npm i
  - 03 - docker compose up -d
  - 04 - npm run setup
  - 05 - npm run dev

#### User registration
- You can add user, email, telephone and date of birth to the system;
- Can remove complete user from the system;
- You cannot leave fields unfilled,
- Receive a confirmation message when creating, deleting or when a field is missing.
